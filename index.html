Audiograph — Guide

Audiograph turns sound into visual art. It converts audio into line-based visualizations that react in real time, with full control over layout, style, color, layering, and particles.

Getting Started

Load audio — click Load Audio to open an audio file (MP3, WAV, etc.). The waveform appears in the timeline at the bottom. Click Play to start the visualization.

Mic / System — capture live audio from your microphone or system audio (Spotify, YouTube, etc.). System audio requires sharing a screen/tab in the browser dialog — check "Share audio".

Timeline — click anywhere on the waveform to seek. Drag to select a range (continuous mode only). Drag the crop handles at the edges to loop a section.

Layers

Everything in Audiograph is a layer. You start with one. Each layer is a fully independent system — its own layout, style, colors, effects, particles, and playback mode. Nothing is shared between layers.

+ Add Layer — creates a new layer with default settings. Each layer has an opacity slider and a remove button (✕). Click a layer name to select it — all panels update to show that layer's settings.

Appearance

Layout — choose the spatial arrangement of lines: Linear, Circle, Kaleidoscope, Spiral, Phyllotaxis, Sine, Concentric, Image, or Custom. Each layout has its own parameters that appear when selected.

Lines / Contrast / Sensitivity — Lines sets how many lines to draw. Contrast controls the dynamic range between quiet and loud. Sensitivity reshapes the audio response curve.

Height / Thickness — line dimensions. Vary Height / Vary Thick — add organic variation. Scale — zoom the layer. Offset X / Y — shift position. Rotation — rotate the layer. Fade Edges — soft falloff at edges. Scale→Loud — scale pulses with audio energy.

Image Layout

Upload any image — it becomes a brightness map. Lines appear where the image is bright (or dark, if Invert is on). Use Threshold to cut the background. Resolution controls the grid density. Direction sets whether lines grow Up, Down, Left, or Right. When Particles are enabled, they spawn from the image pixel positions and carry the actual RGB colors from the image.

Custom Layout

Write JavaScript expressions for x/y positioning. Variables: t (0→1 position), v (loudness), i (index), n (total lines), W/H (canvas size).

Examples: Heart curves, Lissajous figures, rose curves, spirographs, DNA helixes, butterfly curves. See the built-in presets for more.

Line Styles

12 styles: Straight, Rounded, Dotted, Dashed, Pins, Tapered, Caps, Cross, Numbers, Symbols, Words, and Cascade.

Color

Solid — one color for all lines. Tone Ramp — multi-stop gradient mapped to loudness (quiet→loud). Simple Ramp — gradient mapped to position. Click swatches to edit, use presets for quick palettes (14 tone ramp + 14 simple ramp presets), or add/remove stops.

Effects

Playback — Off (hides base visualization), Continuous (waveform pulses with amplitude), or Equalizer (FFT frequency bars). All per-layer.

Particles — toggle On to overlay an audio-reactive particle system. Particles spawn from the layout positions and react to audio energy. See Particle System section below.

Amplify — boost audio response (0.5×–5×). Flip — horizontal and/or vertical per-layer. Mirror — faded vertical reflection. Spin — continuous rotation (circular layouts). Speed — animation speed.

Distortions — Twist (progressive rotation), Bulge (fisheye/pinch), Wave (animated ripple + adjustable frequency), Shear (diagonal skew), Depth (fake 3D perspective).

Drop Switcher — detects quiet→loud transitions and triggers visual effects.

Particle System

Toggle Particles On in the Effects panel to add audio-reactive particles that overlay your visualization. Particles spawn from the actual layout positions — along lines for Linear, from ring positions for Circle, from custom curve points, and from bright image pixels in Image mode.

Shape — Circle, Square, Star, Line (velocity-oriented), Numbers, Symbols, or Words.

Density — how many particles spawn per frame. Speed — velocity, scaled by audio energy. Life — how long particles last (frames). Size — base size, pulses with loudness. Spread — how far from layout positions particles offset. Gravity — vertical pull (negative = float up). Damping — velocity decay per frame.

Effects also affect particles: Twist curls them, Bulge pushes outward/inward, Wave adds sinusoidal drift.

Set Playback to Off + Particles On for a particles-only visualization.

Intensity

The global Intensity slider at the top of the controls (above all panels) multiplies all audio modulation across all layers. Range 0–200%. Use it to quickly boost or reduce the overall audio reactivity.

Randomness

Global — master intensity for all randomness groups. Distribution — randomizes line spacing. Appearance — randomizes height/thickness per line. Outline — randomizes outline offsets. Color — shifts hue per line. Disorder — organic positional jitter.

Re-Roll — generates new random seeds. Auto — continuous sine-wave parameter drift. Random — complete randomization of all settings.

Presets

13 built-in presets (★) covering various layouts and styles. Loading a preset replaces only the active layer — other layers stay untouched.

Save Preset — saves the current state. Load Preset — loads a saved preset. Clear All — removes all user presets.

3D Mode

Toggle 3D to render lines as 3D objects in a WebGL scene. Choose from 5 shapes (Cylinder, Box, Cone, Diamond, Flat) and 6 materials (Standard, Metal, Plastic, Matte, Glass, Neon). Mouse drag to orbit, scroll to zoom.

Fullscreen

Click the expand icon in the bottom-right corner of the preview (visible on hover), or press ⌘F / Ctrl+F. The canvas fills the entire screen — ideal for projection. Press Escape to exit.

Export

Download PNG — saves the current frame (with or without background). SVG — vector export. Record — captures video with audio as MP4/WebM.

Keyboard Shortcuts

⌘/Ctrl + Z Undo   ⌘/Ctrl + F Fullscreen   Space Play/Pause   Esc Exit fullscreen / close help

Tips

Use multiple layers with different layouts and colors for rich compositions. Try the Random button for instant generative exploration. Save presets to keep your favorite setups. For the Image layout, high-contrast images work best — use Threshold to clean up the background. For projection, use Fullscreen mode (⌘F) with a dark background. Combine Particles with Playback Off for a particle-only mode.
